# 🥚 Eggy Study Tracker (CLI Python Edition)

一个轻量、模块化的 Python 学习记录追踪器，专为蛋仔们设计。

## 项目结构
- `app/`: 核心业务逻辑
- `data/`: 存储 SQLite 数据库
- `scripts/`: 数据库初始化及维护脚本

## 快速安装
1. 确保已安装 Python 3.x
2. 本项目使用标准库，无需额外依赖

## 使用方法
在项目根目录下运行：
```bash
python run.py
```

## 功能菜单
1. **Add Study**: 记录你刚刚学习了什么以及时长。
2. **View Today**: 回顾今天的学习列表。
3. **Today Total**: 统计今天一共努力了多少分钟。
